V2 · 蓝雾 #dde4f0 → #cbd5e6
思路：把面板从"灰"挪到品牌靛蓝的最浅层，靠中间调托白。
  • 白色齿轮与中间调底色拉开明确的明度差，"看不清"直接解决；金环在冷蓝上也更醒目
  • 不新增色相 —— 蓝雾就是品牌 indigo 家族的浅端，页面依旧只有蓝 / 绿 / 金三族
  • 为 06 的深靛蓝 CTA 做了"浅蓝 → 深蓝"的铺垫，CTA 仍是全页唯一的深色重锤，节奏顺
  • 另外三个领域组件（白色磨砂背板）放上来是毛玻璃效果，基本成立、只需小调
  • 戏剧性不如 V1；想要"哇"一下的第一眼冲击，它是克制的

V1 · 深靛蓝 #22408e → #0c1a4e
思路：直接用品牌 --brand-indigo-950/800 做一块"珠宝盒"，白齿轮 + 金环对比拉满。
  • 对比最强、最惊艳；金色在深靛蓝上呈首饰级质感，brand echo 也最强
  • 最大问题：06 咨询 CTA 就是深靛蓝渐变大面板。05 再放一块，相邻两屏出现两大块深蓝，CTA 的独占性被稀释 —— 从转化角度，页面里"最深的那块"应该留给行动号召
  • 四个 tab 共用此面板：另外三个领域组件是按浅底设计的，选它就要整组重调（可以做，是个工作量）
  • 04 浅绿 → 深靛蓝的衔接偏硬，需要加过渡带
选它的前提：要么把 06 的 CTA 面板改浅（如香槟金 / 冷白 + 靛蓝按钮），要么接受 05/06 连续两块深蓝的"沉"。

V3 · 浅灰 #eef0f2 → #e6e9ed + 暗角
思路：不换色相，只修"值"。你现在版本真正的问题不是灰，而是面板在齿轮正后方放了一个白色高光斑（radial at 58% 48%）—— 白齿轮叠白光斑，等于主动抹掉图底关系。
  • 高光移弱移开、底色整体压深一档、四周加极轻暗角，白齿轮立刻显形；其他三个组件完全不用动
  • 最安静、零风险，保持你"颜色不要杂"的初衷
  • 依旧谈不上惊艳；如果本节想成为页面记忆点，它不够

结论与落地

  1. 我的推荐：V2 蓝雾。它同时回答了你的两个诉求 —— "看得清"（中间调托白）和"往深蓝系靠"（进入品牌色家族），又不和 06 的深靛蓝 CTA 抢角色。
  2. 真想要 V1 的惊艳：可以，但请连着决定 06 怎么让位（CTA 面板改浅是最干净的解法），并允许我重调另外三个领域组件的配色。
  3. 代码里我已把 V2 设为 .fields__side 的默认背景，V1 / V3 的完整样式以注释块附在同一位置，切换只需替换一段。
  4. 齿轮的金色统一走设计 token --amber-line (#d6a14a)，与 tab 下划线、编号同源。
